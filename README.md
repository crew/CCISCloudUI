# CCISCloudUI
UI for CCISCloudApi

## Background
This is a completely static application that calls API endpoints from the CCISCloudApi project.

## Development
```
npm install
bower install
gulp watch
```
In a different terminal
```
gulp webserver
```

Now make sure the CCISCloudApi project is also running.
